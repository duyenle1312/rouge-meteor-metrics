Install virtual environment

```bash
python -m venv venv
```

Activate virtual env

Window:
```bash
venv\Scripts\activate
```
Linux/Ubuntu:
```bash
. venv/bin/activate
```

Install packages

```bash
pip install -U -r requirements.txt
```

Modify the input: Go to line 168 in run.py, choose a different range

Run the code

```bash
python run.py
```
